# PPII_Library

Project developed to PPII class (SENAC RS).

## Tech Stack
- Node.js (v14.0.0)
- Npm (6.14.4)
- Postgres (^8.7.1")
- Express (^4.17.1)
- JsonWebToken (^8.5.1)
- dotenv (^10.0.0)

## Start the project
- `npm install`
- `npm start` or `nodemon start`
- Local access: http://localhost:3000/
- Heroku: https://library-app-pp2.herokuapp.com/
